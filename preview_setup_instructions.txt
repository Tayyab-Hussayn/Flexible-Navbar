To preview the Navbar component, you need to set up a development environment.
This project is a component library and doesn't include a development server.
Here are the steps to set up a preview using Vite, a fast development server.

1. Install Dependencies:
   Open your terminal and run the following command to install the necessary development dependencies:

   npm install -D vite @vitejs/plugin-react react react-dom @types/react @types/react-dom tailwindcss postcss autoprefixer

2. Create an `index.html` file:
   In the root of your project, create an `index.html` file with the following content:

   <!DOCTYPE html>
   <html lang="en">
     <head>
       <meta charset="UTF-8" />
       <meta name="viewport" content="width=device-width, initial-scale=1.0" />
       <title>Navbar Preview</title>
     </head>
     <body>
       <div id="root"></div>
       <script type="module" src="/src/main.tsx"></script>
     </body>
   </html>

3. Create a `src` directory and move files:
   Create a new directory named `src` in the root of your project. Move the following files into the `src` directory:
   - Navbar.tsx
   - Demo.tsx

4. Create a main entry file:
   Inside the `src` directory, create a new file named `main.tsx` with the following content:

   import React from 'react';
   import ReactDOM from 'react-dom/client';
   import Demo from './Demo';
   import './index.css';

   ReactDOM.createRoot(document.getElementById('root')!).render(
     <React.StrictMode>
       <Demo />
     </React.StrictMode>,
   );

5. Configure Tailwind CSS:
   Create a `tailwind.config.js` file in the root of your project with the following content:

   /** @type {import('tailwindcss').Config} */
   export default {
     content: [
       "./index.html",
       "./src/**/*.{js,ts,jsx,tsx}",
     ],
     theme: {
       extend: {},
     },
     plugins: [],
   }

   Create a `postcss.config.js` file in the root of your project with the following content:

   export default {
     plugins: {
       tailwindcss: {},
       autoprefixer: {},
     },
   }

   Create a new file named `index.css` in the `src` directory with the following content:

   @tailwind base;
   @tailwind components;
   @tailwind utilities;

6. Create a `vite.config.ts` file:
    In the root of your project, create a `vite.config.ts` file with the following content:

    import { defineConfig } from 'vite'
    import react from '@vitejs/plugin-react'

    // https://vitejs.dev/config/
    export default defineConfig({
      plugins: [react()],
    })

7. Update `package.json`:
   Open your `package.json` file and replace the existing `scripts` section with the following:

   "scripts": {
     "dev": "vite",
     "build": "tsc && vite build"
   },

8. Start the development server:
   Now you can start the development server by running the following command in your terminal:

   npm run dev

   This will start a server, and you can view the preview of your Navbar component in your browser at the address provided in the terminal (usually http://localhost:5173).
